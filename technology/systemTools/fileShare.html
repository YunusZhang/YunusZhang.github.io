    # Windonws 文件共享，Mac 访问

    1.首先确定Windowns 和 Mac 是在同一局域网下。

2.Windown 在要共享的目录，右键 》属性 》共享 》高级共享 》勾上“共享此文件夹”》确定。

注意：这一步网上有很多说新建用户组，改权限的，确实很安全，但是和自己需求不符；这里直接把文件夹共享，然后远程登录win7账户，用此账户去操作共享文件夹就行了；


3.Mac 打开Finder 》前往 》连接服务器 》输入“smb://windows的ip地址”，然后使用 Windows 的账号密码登录，就可以看到共享的文件夹了。

首先，cmd-->ipconfig获取windows的ip地址，注意选择正确的ip;

 Windows 的账号密码登录administrator, 密码自己登录win7的密码
 
4.注意：此方式只能在win7上操作文件，想要在mac上传输文件到共享文件夹（或者写入执行），目前还未实现，以后有时间再尝试
