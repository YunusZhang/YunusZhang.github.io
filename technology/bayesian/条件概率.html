      # 1.1 观察和信念

      信念描述了我们所知道的世界,当观察到某件事时,它的条件概率就代表了在我们相信的前提下自己所见事情的可能性,即

      P(观察 \mid 信念)

      P(B∣A)P(B \mid A)P(B∣A)

      例如:如果我相信气候变化是真的,那么观察到10年发生5次干旱的概率有多大?

      上面可能性的反转,有没有某种方法来量化自己有多相信气候真的在发生变化;也就是求解出在给定观察的情况下,信念的概率, 即

      P(信念 \mid 观察)

      P(A∣B)P(A \mid B)P(A∣B)

      # 1.2 小结

      贝叶斯定理允许我们将对世界的信念和数据结合起来,然后根据我们观察到的情况把这种结合转化为对信念强度的估计.
