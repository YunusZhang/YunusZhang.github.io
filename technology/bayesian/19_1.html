# 贝叶斯思维

P(A∣B)P(A \mid B)P(A∣B)

ab\frac{a}{b}​b​​a​​ a>ba>ba>b

# 1 日常推理

# 1.1 先验信念和条件概率

先验信念: 根据我们一生的经验(也就是观察到的数据)建立起来的信念集合. 在公式中先验信念在数据后面并用|与数据隔开

P(窗外出现亮光，天空中有蝶形物体|地球上的经验)=很小

上面的公式可以理解为:根据地球的经验,在天空中同时看到亮光和蝶形物体的概率很小.

这个概率结果被称为条件概率

P(D∣X)P(D \mid X)P(D∣X)

D=窗外出现亮光，天空中有蝶形物体(数据)
X=地球上的经验(先验信念)

多重信念为条件
P(窗外出现亮光，天空中有蝶形物体|特定节日,地球上的经验)=小

# 1.2 假设

当看到数据并认为自己看到了UFO时,你就在形成一个假设,UFO的假设很可能基于你以前看过的电影和电视节目,将第一个假设定义为: H1=在我家后院有一个UFO!

那这个假设预测又是什么呢?倒过来想,如果在你家后院有一个UFO,那么你预期会看到什么呢?你可能会回答:亮光和蝶形物体.因为H1预测了数据D,所以当我们在给定的假设下观察到数据时,数据的概率就会增加.

P(D∣H1,X)P(D \mid H1,X)P(D∣H1,X) >>P(D∣X)P(D \mid X)P(D∣X)
上面的意思:如果相信这是UF并根据经验,在天空中看到亮光和蝶形物体的概率要远远大于只看到亮光和蝶形物体而无法解释的概率

# 1.3 搜集更多的数据以更新信念

最初假设: H1=有UFO着陆! P(H1∣X)P(H1 \mid X)P(H1∣X)=非常小

当观察到更多数据后,你意识到还有个可能的假设 H2=有人在窗外拍电影

P(H2∣X)P(H2 \mid X)P(H2∣X)=很小

当改变信念时,我们需要用一种方法将新得到的数据考虑进去

# 1.4 备择假设

思考备择假设的过程就是利用你所掌握的数据对多种假设进行比较的过程

D更新后=亮光,蝶形物体,电线,摄制组,其他灯光等

P(D更新后 \mid H1,X)=非常小

你有了一种新的假设H2,可以更恰当的解释数据 P(D更新后 \mid H2,X) >> P(D更新后 \mid H1,X)

这里的关键是,要理解我们是在比较这些假设对观测数据的解释程度. 远远大于表示第二种假设可以更恰当的解释所观察到的数据.

由此,贝叶斯分析的真正核心:检验信念的标准是他们解释世界的能力 我们说一种信念要比另一种信念更准确,因为他能更恰当的解释我们所观察到的世界

\frac{P(D更新后 \mid H2,X)}{P(D更新后 \mid H1,X)}

(注意符号只是公式分界符,在两个符号里面不需要再出现$分界符)

当这个比值是一个很大的值比如1000时,它意味着H2对数据的解释要比H1恰当1000倍

# 1.5 数据影响信念,信念不应该影响数据

强调:所有这些实例中唯一不变的是数据,你的假设可以改变,你在世界上的经验X也可以与别人不同,但是数据D则是所有人共享的

P(D∣H,X)P(D \mid H,X)P(D∣H,X) 可以理解为:根据给定的假设和我的经验所得出的数据概率,或者我的信念对所观察到的数据解释得如何

P(H∣D,X)P(H \mid D,X)P(H∣D,X) 可以理解为:数据和在这个世界上的经验,我的信念的概率,或者我观察到的情况对我的信念的支持程度

在第一种情况下我们会根据所收集到的数据和对这个世界的观察来改变自己的信念,从而更恰当地描述这个世界.第二种情况下,我们收集数据来支持自己当前的信念.

贝叶斯思维就是改变你的想法,更新你对世界的理解. 我们观察到的数据都是真实的,所以我们的信念终归需要转变,直到与数据一致

# 1.6 小结

你根据现有的经验X形成了最初的信念.而观察到的数据D,要么与你的经验一致,即P(D∣X)P(D \mid X)P(D∣X)=很大; 要么让你感到惊讶,即P(D∣X)P(D \mid X)P(D∣X)=很小. 为了理解这个世界,你信赖根据观察所形成的信念,或者说假设H. 很多时候,一种呢新的假设可以解释让你感到惊讶的数据,用概率的语言表示就是P(D∣H,X) >>P(D \mid X).

当收集到新的数据或者产生新的想法时,你可以形成更多的假设,如H1,H2,H3等.当一种新的假设要比旧的假设更能解释收集到的数据时,即当出现下面情况时,你会改变信念

P(D∣H2,X)P(D∣H1,X)\frac{P(D \mid H2,X)}{P(D \mid H1,X)}​P(D∣H1,X)​​P(D∣H2,X)​​=较大的值

最后你应该更关注哪些改变你信念的数据,而不是确保数据支持你的信念,即P(H∣D)P(H \mid D)P(H∣D)的值
