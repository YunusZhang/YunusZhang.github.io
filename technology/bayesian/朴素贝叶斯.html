              # 1.1 公式

              T={(x1,y1),(x2,y2)...(xn,yn)}

              x=(x1,x2,,,xn)

              yi=ck, 其中k=1,2,,,k

              给定的数据集T, 其中的x样本特征, 由一系列独立的特征维度构成; y是样本的类别, 可能属于c1到ck中的任何一个

              朴素贝叶斯就是根据样本特征x, 预测样本属于的类别y, P(yi=ck∣x)=P(yi=ck)∗P(x∣yi=ck)∑i=1nP(yi=ck)∗P(x∣yi=ck)P(yi=ck \mid x)=\frac{P(yi=ck)*P(x|yi=ck)}{\sum_{i=1}^{n}P(yi=ck)*P(x \mid yi=ck)}P(yi=ck∣x)=​∑​i=1​n​​P(yi=ck)∗P(x∣yi=ck)​​P(yi=ck)∗P(x∣yi=ck)​​

              得到正比关系

              P(yi=ck∣x)P(yi=ck \mid x)P(yi=ck∣x)正比于P(yi=ck)∗P(x∣yi=ck){P(yi=ck)*P(x|yi=ck)}P(yi=ck)∗P(x∣yi=ck)

              那么就是取后面项的最大值

              因为我们假设x样本特征是相互独立的, 所以计算类别的可能性就归结于一系列独立事件概率相乘的(条件概率的相乘), 也就是每个类别的可能性乘以基于此类别下是样本x的可能性

              最后取最大的那个值;

              最后选择正确程度最高的类别作为分类的结果;

              注:上面是连乘符号

              # 1.2 范例

              y1= P(书信)\times P(玫瑰|书信)\times P(想念|书信) y2= P(报告)\times P(玫瑰|报告)\times P(想念|报告)

              ...

              在y1,y2,,,中取正确程度最高的

              在朴素贝叶斯中, 假设每个特征维度之间是相互独立的; 在例子中认为关键词之间是相互独立的, 也就是说他们出现的顺序和上下文关系并不影响计算结果, 哪怕他们之间表达的意思千差万别;

              注意,朴素贝叶斯中是以一系列独立的特征维度来比较其属于不同类别的可能性

              # 1.3 具体步骤和样本集

              1.搜集所有的样本集的类别, 并计算每个类别(标签)的概率;

              2.然后统计样本中的特征维度, 计算每个特征维度在对应类别中出现的概率;

              3.统计目标集中的特征维度; 分别计算出目标集中特征维度所属类别的可能性;

              4.取其中正确程度最高的;

              # 1.4 小问题 拉普拉斯平滑技巧

              如果某个特征维度没有出现在样本集类别中, 会导致对应特征维度在此样本集中概率为0, 显然这样会误判;

              就需要人为的给每个特横维度加1, 避免出现概率为0的情况;

              # 1.5 实际应用中

              在分类任务中通常用来做文本数据的分类, 如, 垃圾邮件的过滤, 文档的自动分类, 情感分析, 此外还有多分类实时预测, 推荐系统(贝叶斯和协同过滤组合使用), 拼写校正

              实际中很难出现特征维度之间相互独立的情况, 假如有n个特征, 那这n个特征之间是否有关联呢, 是否独立呢?

              需要我们对特征维度数据做预处理, 如选择合适的特征维度, 降维等手段降低他们之间的耦合

              # 1.6 高斯模型

              当样本为例如身高体重等类似的连续特征, 并且样本数量太少无法进行区间计算时, 我们可以根据样本计算出均值和方差,并且假设样本分布为正态分布, 这样我们就得到密度函数. 根据密度函数, 我们可以将待求的特征值代入, 得到密度函数的值. 比如, 男性的身高均值是5.855、方差0.035的正太分布. 所以,所以, 男性的身高为6英尺的概率的相对值为1.5789, 根据得到的值进行计算类别.
