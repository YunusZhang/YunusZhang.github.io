        # 贝叶斯公式和全概率公式

        # 1.1 贝叶斯公式

        一般的贝叶斯公式是如下的形式

        P(A∣B)P(A \mid B)P(A∣B)=P(B∣A)∗P(A)P(B)\frac{P(B \mid A)*P(A)}{P(B)}​P(B)​​P(B∣A)∗P(A)​​

        但是这就涉及到一个问题,P(B)作为数据,如何把它归一化的问题,

        思路一是不考虑P(B),反正P(A∣B)P(A \mid B)P(A∣B)和P(B∣A)∗P(A){P(B \mid A)*P(A)}P(B∣A)∗P(A)成正比, 直接拿后者的不同结果去做比值, 直接看比值结果的大小;

        思路二是,既然要求出每条单独路径的可能性,那么能不能直接拿每条单独路径的除以所有路径的总和,即:

        P(Aj∣Bj)P(Aj \mid Bj)P(Aj∣Bj)=P(Aj)∗P(Bj∣Aj)∑j=1nP(Aj)∗P(Bj∣Aj)\frac{P(Aj)*P(Bj \mid Aj)}{\sum_{j=1}^{n}P(Aj)*P(Bj \mid Aj)}​∑​j=1​n​​P(Aj)∗P(Bj∣Aj)​​P(Aj)∗P(Bj∣Aj)​​

        注意:上面的错误,B只是第二步的结果,不存在Bj;故更改如下

        P(Aj∣B)P(Aj \mid B)P(Aj∣B)=P(Aj)∗P(B∣Aj)∑j=1nP(Aj)∗P(B∣Aj)\frac{P(Aj)*P(B \mid Aj)}{\sum_{j=1}^{n}P(Aj)*P(B \mid Aj)}​∑​j=1​n​​P(Aj)∗P(B∣Aj)​​P(Aj)∗P(B∣Aj)​​

        # 1.2 全概率公式

        解释:P(B)发生的概率就是,所有路径之和

        P(B)=∑j=1nP(Aj)∗P(Bj∣Aj)\sum_{j=1}^{n}P(Aj)*P(Bj \mid Aj)∑​j=1​n​​P(Aj)∗P(Bj∣Aj)

        注意:上面的错误,B只是第二步的结果,不存在Bj;故更改如下

        P(B)=∑j=1nP(Aj)∗P(B∣Aj)\sum_{j=1}^{n}P(Aj)*P(B \mid Aj)∑​j=1​n​​P(Aj)∗P(B∣Aj)
