    # 联想G50-80添加内存条

    # 1 规格

    联想G50 80 4G内存可以另外加装一个内存条的。

    联想G50 80 内存规格如下：

    内存容量：4GB（4GB×1）；

    内存类型：DDR3L 低压版度；

    插槽数量：2×SO-DIMM；

    最大内存容量：16GB；

    扩展内存的话回，可以考虑加一个 DDR3L - 1600 4G（ 或 8G） 的笔记本内存。加装前取下笔记本的电池和电源线，然后用梅花螺丝刀拆答掉后壳加装即可。

    # 2 注意事项

    # 2.1 不对称双通道对性能的影响

    4+8会被识别为一个4+4的双通，多余4g会是单通；8+4直接就被识别为一个残缺的双通，所以比单通快不了多少

    所以靠近cpu的卡槽安装4g内存，远离cpu的卡槽安装8g内存；也可以看cpuz，插槽1插4g，插槽2插8g

    # 2.2 安装注意

    别硬来，对准缺口，卡扣缺口一定要怼到底再按下去；注意要听到的是咔哒一声，不是咔嚓一声哦！
